# Foundation models
This respository contains implementations of frameworks for foundatio models, starting with an improved version of the clinical transformer. 

## Clinical Transformer
Implementation of the clinical transformer for large datasets. Highly modular for integrating new architectures. This current version has implemented:

* classification 
* transfer learning
* masked self supervised learning
* regression
* Survival analysis
* -[PBMF (predictive biomarker modeling framework) loss]
